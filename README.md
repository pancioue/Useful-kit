# Useful-kit

必選
--------
* PHP Intelephense 
  - Rapid workspace wide go to definition support.
  - Workspace wide find all references.
  - 未被引用到的變數，會變暗色提示

推薦
--------
* Git Graph 
  - git 圖
* GitLens
  - 查看最後是誰編輯
  - gitlens inspect 可查看歷史修正紀錄
* Code Spell Checker 
  - 修正單字拼錯，拼錯會顯示須須
 
選用
--------
* PHP Namespace Resolver  (選用)
  - 自動引用class，整理class
* Trailing Spaces  (選用)
  - 自動去掉尾部多餘的空白
